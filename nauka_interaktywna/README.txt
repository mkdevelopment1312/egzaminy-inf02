# Nauka Interaktywna - Struktura Folderów

Ten folder zawiera materiały do nauki interaktywnej, które będą wyświetlane w sekcji "Nauka Interaktywna" na stronie.

## Struktura folderów

- `nauka_interaktywna/` - Główny folder dla materiałów do nauki interaktywnej
  - `media/` - Folder na pliki multimedialne (obrazy, filmy, itp.)
  - `README.txt` - Ten plik

## Jak dodawać materiały

1. Umieść pliki multimedialne w folderze `media/`
2. Utwórz pliki tekstowe z treścią materiałów w głównym folderze `nauka_interaktywna/`
3. Zaktualizuj kod strony, aby wyświetlać nowe materiały

## Format plików

Pliki tekstowe powinny być w formacie Markdown lub HTML, aby można było łatwo formatować tekst.

## Przykład struktury materiału

```
# Tytuł materiału

## Podtytuł

Treść materiału...

![Opis obrazka](media/nazwa_pliku.jpg)

## Kolejny podtytuł

Więcej treści...
```
